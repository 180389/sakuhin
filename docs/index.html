ドラゴンクエスト11

完成度の高いオープンワールドの世界を自由に旅することができます。メインストーリーのボリュームは結構あって全てクリアすると満足感が得られます。ミニゲームも多くてメインストーリーをクリアした後でも何時間でも遊ぶことができます。今までのドラゴンクエストシリーズをやっていた人ならより楽しめる要素が各所に散りばめられています。swicthでも今秋にドラゴンクエストSというタイトルで発売されます。変更点はキャラクターに声がつく、各キャラクターのミニストーリーが楽しめるなどです。少しでも興味を持っていただけたなら購入してみる事をオススメします。

モンスターストライク

スマホを持っている人なら一度は見たことがあるであろうと思うくらい人気なゲームです。ゲームの内容は簡単にいうとおはじきです。ただおはじきと違うのがおはじき同士がぶつかると「友情コンボ」と呼ばれる様々な技がでます。それを駆使してステージクリアを目指すゲームです。楽しい部分が、こんなステージよく思いつくなというくらい様々なギミックの難しいステージが時間で配信されていて、難しいステージを何回もやり直しながらやっとクリアしたときの達成感が楽しいです。さらにマルチプレイにも対応していて近くの友達とわいわいプレイすることや専用のアプリを使えば遠くの人と一緒にプレイすることもできます。スマホを持っていれば無料で始められるのでぜひプレイしてみてはいかがしょうか。

大乱闘スマッシュブラザーズSP

有名な対戦ゲームです。対戦ゲームでダメージを与えるとパーセンテージが溜まっていき、パーセンテージが高いほどステージの外に出されやすくなります。最終的に多くステージの外に出した人の勝利というルールです。このゲームの面白いところがキャラクターの操作性です。自由に操作することができるので、上手いトッププレイヤーはまるで自分の手足のように操作できます。e-sportsとしても有名で賞金つきの世界大会が開催されるほどなので、賞金狙いで始めてみてもいいかもしれません.
